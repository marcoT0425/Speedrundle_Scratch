# Speedrundle_Scratch
Speedrun version of wordle

Speedrundle is a speedrun version of Wordle, which is based on Hello Wordl and Speedle (https://tck.mn/speedle/)

In the first release, there are several features for accessibility, eg the tiles have "tick" or "question mark" markers or the high contrast mode for players with low vision. I mean, there is also a profanity filter to censor swear words and slurs.
