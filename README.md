# Speedrundle_Scratch
Speedrun version of wordle

Speedrundle is a speedrun version of Wordle, which is based on Hello Wordl and Speedle (https://tck.mn/speedle/)

# Features

My speedrundle is a bit better than the one created by tck.mn. For the tiles, they have a "tick" sign to indicate a correct letter while a "question mark" indicates a misplaced letter. This is designed for accessibility with people of low vision of those having dyslexia.

I've also added a profanity filter to censor swear words so that it can prevent children saying or writing these kind of language. I mean, I've uploaded the list and warned people not to open it!

If the game has been ended, the statistics will be automatically copied to the clipboard. You may share it with using social media eg Reddit or Discord.

Should there be any problems, feel free to ask
